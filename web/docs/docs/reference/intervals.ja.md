# インターバル一覧

PalmScript は現在の Binance kline インターバルをすべてサポートします。

| Literal | Meaning |
| --- | --- |
| `1s` | 1 秒 |
| `1m` | 1 分 |
| `3m` | 3 分 |
| `5m` | 5 分 |
| `15m` | 15 分 |
| `30m` | 30 分 |
| `1h` | 1 時間 |
| `2h` | 2 時間 |
| `4h` | 4 時間 |
| `6h` | 6 時間 |
| `8h` | 8 時間 |
| `12h` | 12 時間 |
| `1d` | 1 日 |
| `3d` | 3 日 |
| `1w` | 1 週 |
| `1M` | 1 か月 |

大文字小文字に関する注意:

- 週足は `1w`
- 月足は `1M`
