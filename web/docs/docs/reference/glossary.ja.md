# 用語集

## ベースインターバル

`interval <...>` で宣言される実行インターバル。

## 宣言済みインターバル

名前付きソースに対して `use <alias> <...>` で明示的に導入されたインターバル。

## ソース対応スクリプト

少なくとも一つの `source` を宣言するスクリプト。

## ソーステンプレート

`binance.spot` や `hyperliquid.perps` のような組み込みの取引所 / venue コンストラクタ。

## マーケットモード

履歴市場フィードに対する実行。

## ノールックアヘッド

上位インターバルのサンプルは、そのローソク足が完全に確定した後でのみ見えるという保証。

## 出力シリーズ

`export` または `trigger` によって出力される、名前付きの各ステップ結果。

## ベースタイムスタンプの和集合

宣言された全ソースのベースインターバルのローソク足始値時刻の和集合から構築される実行タイムライン。
