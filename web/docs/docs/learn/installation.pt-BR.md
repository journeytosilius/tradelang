# Instalacao

PalmScript e demonstrado publicamente no momento por meio do IDE hospedado no navegador:

[https://palmscript.dev/app/](https://palmscript.dev/app/)

Antes de seguir os exemplos de `Aprenda`, certifique-se de que:

- voce consegue acessar a app hospedada
- seu navegador permite JavaScript e WebAssembly
- voce consegue editar e executar scripts dentro do IDE

Os exemplos publicos de `Aprenda` assumem que voce esta escrevendo PalmScript
dentro do IDE do navegador, revisando os diagnosticos ali e executando o fluxo
de backtest de BTCUSDT pela app.
