# インストール

PalmScript は現在、ホストされたブラウザ IDE を通じて公開デモされています。

[https://palmscript.dev/app/](https://palmscript.dev/app/)

`学ぶ` の例を進める前に、次を確認してください。

- ホストされたアプリにアクセスできる
- ブラウザで JavaScript と WebAssembly が有効になっている
- IDE の中でスクリプトを編集して実行できる

公開されている `学ぶ` の例は、ブラウザ IDE の中で PalmScript を書き、
そこで診断を確認し、アプリから BTCUSDT の履歴バックテストを実行すること
を前提にしています。
