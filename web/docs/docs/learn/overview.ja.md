# PalmScript を学ぶ

PalmScript の公開ドキュメントは次の二つを中心に構成されています。

- 戦略を書くための言語
- スクリプトの書き方と使い方を示す例

## PalmScript で行うこと

典型的な流れ:

1. `.ps` スクリプトを書く
2. ベースの `interval` を宣言する
3. 1 つ以上の `source` バインディングを宣言する
4. ブラウザ IDE で検証する
5. アプリ内で履歴データに対して実行する
