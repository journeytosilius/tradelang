# Installation

PalmScript est actuellement presente publiquement via l'IDE navigateur heberge :

[https://palmscript.dev/app/](https://palmscript.dev/app/)

Avant de suivre les exemples de `Apprendre`, assurez-vous que :

- vous pouvez acceder a l'application hebergee
- votre navigateur autorise JavaScript et WebAssembly
- vous pouvez modifier et executer des scripts dans l'IDE

Les exemples publics de `Apprendre` supposent que vous ecrivez PalmScript dans
l'IDE navigateur, que vous y consultez les diagnostics et que vous lancez le
flux de backtest BTCUSDT depuis l'application.
