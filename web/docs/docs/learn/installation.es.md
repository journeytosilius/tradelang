# Instalacion

PalmScript se demuestra publicamente por ahora a traves del IDE alojado en el navegador:

[https://palmscript.dev/app/](https://palmscript.dev/app/)

Antes de seguir los ejemplos de `Aprende`, asegurate de que:

- puedes acceder a la app alojada
- tu navegador permite JavaScript y WebAssembly
- puedes editar y ejecutar scripts dentro del IDE

Los ejemplos publicos de `Aprende` asumen que escribes PalmScript dentro del
IDE del navegador, revisas ahi los diagnosticos y ejecutas el flujo de backtest
de BTCUSDT desde la app.
